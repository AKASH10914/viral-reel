## ⚡ Quick Start (2 minutes)

### 1. Install & Run
```bash
npm install
npm run dev
```

### 2. Access
- Main App: http://localhost:3001
- Admin Panel: http://localhost:3001/admin
- Admin Token: `admin-test-secret-key-12345` (header: `x-admin-token`)

### 3. Test All Features
```bash
node test-features.mjs
# Output: 9/9 PASS ✅
```
